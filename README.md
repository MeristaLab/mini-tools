# mini-tools

日常的に使えるミニツールをまとめたリポジトリです。

## ツール一覧

| ツール名 | 説明 |
|---------|------|
| [qr-generator](./qr-generator/) | URL・テキストからQRコードを生成するGUIツール |
| [heic2jpeg](./heic2jpeg/) | HEICファイルをまとめてJPEGに変換するGUIツール |

## 使用技術

HTML / CSS / JavaScript / Python（ツールによる）
