## 作成候補（優先度S：まず作る候補）

- [ ] PDF結合ツール：複数PDFを選択または指定して、順番に1つのPDFへ結合する。まずは動けばOK。
- [ ] Dropbox整理レポート群：1つのフォルダに複数スクリプトをまとめる方式。削除はせず、状態確認だけにする。
- [ ] フォルダ容量ランキング：指定フォルダ内の容量が大きいフォルダ上位5〜10件を表示。
- [ ] ファイル容量ランキング：指定フォルダ内の容量が大きいファイル上位5〜10件を表示。
- [ ] フォルダ数・ファイル数カウント：指定フォルダ配下の総フォルダ数・総ファイル数・総容量を表示。
- [ ] 最近更新されたフォルダ一覧：更新日時が新しいフォルダ上位5〜10件を表示。
- [ ] 1年以上更新なしフォルダ一覧：古いフォルダ上位5〜10件を表示。

## 作成候補（優先度A：次に作る候補）

- [ ] フォルダ健康診断レポート：Sの各結果をまとめて、ターミナルに一括表示する。
- [ ] HTMLレポート出力：ターミナル結果をHTMLとして保存し、一般の人にも見せやすくする。
- [ ] CSV出力：容量ランキングや更新日時一覧をCSVに保存する。
- [ ] 拡張子別容量集計：動画・画像・PDF・ZIPなど、種類ごとの容量を表示。
- [ ] 古い大容量ファイル候補：容量が大きく、かつ長期間更新されていないファイルを表示。
- [ ] キャッシュ・一時ファイル候補表示：cache、temp、preview、renderなどの名前を含むフォルダを検出する。

## 作成候補（優先度B：余裕があれば作る候補）

- [ ] 重複ファイル候補表示：同一内容のファイルをハッシュで検出。ただし削除はしない。
- [ ] 同名ファイル一覧：同じ名前のファイルを一覧化。ただし判断が難しいので優先度は低め。
- [ ] 命名ルール違反チェック：final、完成版、最新版、修正済みなど、混乱しやすい名前を検出。
- [ ] 案件フォルダ構成チェック：素材、書き出し、音声、画像などの標準フォルダがあるか確認。
- [ ] Premiereリンク切れ確認補助：`.prproj`解析は不確実なので、将来の調査候補。
- [ ] 未使用ファイル候補表示：安全判定が難しいため、削除ではなく参考情報として扱う。

## 方針メモ

- 最初は**ターミナル表示**で作る。
- 1ファイル1機能でも、1フォルダにまとめて管理する。
- 削除機能は最初から入れない。
- 目的は「整理」ではなく**状態の見える化**。
- 名前は仮で **Dropbox整理レポート** または **フォルダ健康診断ツール群** がよさそう。
