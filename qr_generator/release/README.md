# QR Generator

URLやテキストからQRコードを生成するツールです。

## 使い方

1. `qr_generator.exe` をダブルクリックで起動
2. URL またはテキストを入力して「生成」ボタンを押す（Enterキーでも可）
3. 色・サイズ・余白・保存形式を必要に応じて設定
4. 「ダウンロードフォルダに保存」ボタンで保存

## 機能

- 前景色・背景色をカラーピッカーまたはHEXコードで自由に設定
- サイズ：S（256px）/ M（512px）/ L（1024px）
- 保存形式：PNG / JPEG
- 保存先：ダウンロードフォルダに自動保存
- 同名ファイルが存在する場合は上書き確認ダイアログを表示

## 動作環境

Windows 10 / 11
