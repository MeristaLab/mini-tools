# HEIC2JPEG

HEICファイルをまとめてJPEGに変換するツールです。

## 使い方

1. `HEIC2JPEG.exe` をダブルクリックで起動
2. フォルダ選択ダイアログが開くので、HEICファイルが入ったフォルダを選択
3. 変換が完了すると、選択したフォルダ内に `jpeg_output/` フォルダが作成され、JPEGファイルが保存されます

## 注意事項

- 同名ファイルが存在する場合は `_2`, `_3` と連番を付けて保存します（上書きなし）
- 保存品質は92（高画質）固定です

## 動作環境

Windows 10 / 11
